# Flutgraben Website — Sitemap

## Navigation structure

Home page + 5 main sections + Impressum (footer). Language switcher in global header (default: English).

---

## Accueil — Main Page

**Concept:** Entry point of the site. Introduces Flutgraben — the space, its identity, and its programming. Leads visitors toward the main sections.

---

## Spree

**Concept:** Events and programming for the current year.
The name references the river Spree — events flow through time like water.

| Subpage        | Scope                                           |
|----------------|-------------------------------------------------|
| Upstream       | Past events from the current year               |
| In Flutgraben  | Current month                                   |
| Downstream     | Upcoming months within the current year         |
| Archive        | Past years — dropdown to select a year          |

**Navigation pattern:** segmented subnav (button group). Standard component reused across all interior pages.

---

## Eddies

**Concept:** Inspired by quantum physics — eddies (tourbillons) as a metaphor for what Flutgraben generates and offers. Everything that orbits the space.

| Subpage    | Content                                                                 |
|------------|-------------------------------------------------------------------------|
| Overview   | What Flutgraben offers — general introduction                           |
| Moons      | How to join / gravitate around Flutgraben — like satellites in orbit    |
| Canteen    | Tuesday canteen cooked by an artist in residence for the whole community|
| Rhizotype  | Typography developed by and for Flutgraben — typeface presentation      |
| Friends    | TBD — partner organizations, friendly structures, collaborators         |
